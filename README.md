<p>Factions of the Forged Seas is a magic-fueled fantasy factions modpack with a Server set in a world of arcane warfare, mythical creatures, and high-seas conquest. Build your faction's stronghold, master Thaumcraft, Blood Magic, and Ars Magica, and harness forbidden relics to gain an edge over rival warlords. 
Siege enemy bases, wield legendary weapons from Tinkers' Construct, and summon devastating forces with Witchery. With customized world generation, economy systems, and naval battles, every expedition is a test of survival, diplomacy, and domination.</p> 
<p><strong>Will your faction rise as rulers of the arcane seas, or be lost to the depths of history?</strong></p>

# Build:

This modpack uses https://github.com/juraj-hrivnak/Pakku as package/mod managing tool
```sh
# Inside project folder
pakku export
# It will produce curseforge and modrinth format modpacks inside build folder
# If you want archive with all mods you need to write:
pakku fetch
```
